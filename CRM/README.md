# How to run
## Using Visual Code
- Open Visual Code IDE
- Click Terminal in Menubar -> Select New Terminar(Or using hotkey(MacOS): "control + `")
## Run using command
- Run the `go get github.com/gorilla/mux`
- Run the `go run .`
